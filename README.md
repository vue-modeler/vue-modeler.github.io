# Vue Modeler Documentation

Документация для Vue Modeler - мощной библиотеки для управления состоянием в Vue.js приложениях.

## Структура документации

### Начало работы
- **Введение** - обзор архитектуры и основных концепций
- **Быстрый старт** - пошаговое руководство по созданию первой модели
- **Установка** - инструкции по установке и настройке

### Основы
- **Модели** - создание и использование реактивных моделей
- **Действия** - работа с асинхронными действиями
- **Состояния** - управление состояниями действий
- **Декораторы** - использование декоратора @action

### Продвинутые темы
- **Обработка ошибок** - детальная обработка ошибок
- **Отмена операций** - работа с AbortController
- **Блокировка действий** - предотвращение повторных вызовов
- **Жизненный цикл** - управление ресурсами и состоянием

### Примеры
- **Простой пример** - счетчик с демонстрацией основных возможностей
- **CRUD операции** - полноценное приложение для управления пользователями

### Справочник
- **model** - функция создания провайдера модели
- **ProtoModel** - базовый класс для всех моделей
- **Action** - класс для управления действиями
- **@action** - декоратор для объявления действий

## Разработка документации

### Установка зависимостей

```bash
pnpm install
```

### Запуск в режиме разработки

```bash
pnpm dev
```

### Сборка для продакшена

```bash
pnpm build
```

### Предварительный просмотр сборки

```bash
pnpm preview
```

## Технологии

- **Astro** - статический генератор сайтов
- **Starlight** - тема документации для Astro
- **TypeScript** - типизация
- **Markdown/MDX** - контент

## Вклад в документацию

1. Создайте новую ветку для изменений
2. Внесите изменения в соответствующие MDX файлы
3. Проверьте локально с помощью `pnpm dev`
4. Создайте pull request с описанием изменений

## Стиль написания

- Используйте русский язык для основного контента
- Пишите понятно и структурированно
- Включайте примеры кода для всех концепций
- Добавляйте ссылки на связанные темы
- Используйте TypeScript для всех примеров кода

## Лицензия

Документация распространяется под той же лицензией, что и Vue Modeler.
