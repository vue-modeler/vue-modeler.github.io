# Vue Modeler Documentation

Документация для Vue Modeler - мощной библиотеки для управления состоянием в Vue.js приложениях.

## Технологии

- **VitePress** - генератор статической документации на основе Vite и Vue 3
- **Vue 3** - для компонентов и реактивности
- **TypeScript** - типизация
- **Markdown** - контент документации

## Разработка документации

### Установка зависимостей

```bash
pnpm install
```

### Запуск в режиме разработки

```bash
pnpm dev
```

Документация будет доступна по адресу `http://localhost:5173`

### Сборка для продакшена

```bash
pnpm build
```

### Предварительный просмотр сборки

```bash
pnpm preview
```

## Структура файлов

```

├── .vitepress/
│   ├── config.ts           # Конфигурация VitePress

│   └── ...
```

## Кастомные компоненты

В документации используются кастомные компоненты:

- `<Card>` - карточка с заголовком и содержимым
- `<CardGrid>` - сетка карточек (с опцией `stagger` для смещения)

Компоненты автоматически регистрируются глобально через `.vitepress/theme/index.ts`

## Вклад в документацию

1. Создайте новую ветку для изменений
2. Внесите изменения в соответствующие Markdown файлы
3. Проверьте локально с помощью `pnpm dev`
4. Создайте pull request с описанием изменений

## Стиль написания

- Используйте русский язык для основного контента
- Пишите понятно и структурированно
- Включайте примеры кода для всех концепций
- Добавляйте ссылки на связанные темы
- Используйте TypeScript для всех примеров кода
