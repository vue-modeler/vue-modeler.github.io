## Что такое Vue Modeler?

Vue Modeler - это современная библиотека для управления состоянием в Vue.js приложениях. Она предоставляет элегантный способ создания реактивных моделей с поддержкой асинхронных действий, состояний и обработки ошибок.

## Быстрый пример

```typescript
import { ProtoModel } from 'vue-modeler'
import { action } from 'vue-modeler/decorator'

class UserModel extends ProtoModel {
  users = []
  loading = false

  @action async fetchUsers(): Promise<void> {
    this.loading = true
    try {
      const response = await fetch('/api/users')
      this.users = await response.json()
    } finally {
      this.loading = false
    }
  }

  @action async createUser(userData: any): Promise<void> {
    const response = await fetch('/api/users', {
      method: 'POST',
      body: JSON.stringify(userData)
    })
    const newUser = await response.json()
    this.users.push(newUser)
  }
}
```

## Почему Vue Modeler?

- **Простота**: Минимальный API для быстрого старта
- **Типизация**: Полная поддержка TypeScript
- **Производительность**: Оптимизированная реактивность с Vue 3
- **Гибкость**: Легкая интеграция с существующими проектами
- **Надежность**: Обширное покрытие тестами
