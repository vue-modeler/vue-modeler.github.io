**@vue-modeler/model** использует **@vue-modeler/dc** для хранения моделей.
Поэтому сначала [установите **@vue-modeler/dc**](https://github.com/vue-modeler/dc?tab=readme-ov-file#installation).

**@vue-modeler/model** не требует дополнительных настроек.
Просто установите пакет.


<Tabs>
  <TabItem label="npm">
    ```bash
    npm install @vue-modeler/model
    ```
  </TabItem>
  <TabItem label="yarn">
    ```bash
    yarn add @vue-modeler/model
    ```
  </TabItem>
  <TabItem label="pnpm">
    ```bash
    pnpm add @vue-modeler/model
    ```
  </TabItem>
</Tabs>
